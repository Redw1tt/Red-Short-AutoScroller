 [ INSTALLATION NOTES ]

  1. Extract the archive into a folder of your choice.
  2. Open your CHROMIUM-based browser.
  3. Type 'chrome://extensions' in the address bar.
  4. Enable [ DEVELOPER MODE ] in the top right corner.
  5. Click on [ LOAD UNPACKED ] (Load extension).
  6. Select the folder and confirm.

 [ GAMING & BACKGROUND SETUP ]

  To prevent the browser from "freezing" the script while you play:

  1. DO NOT MINIMIZE THE WINDOW: Leave Chrome/Brave open behind your game.
     If you minimize it to the taskbar, Windows will suspend the script.
  2. DISABLE MEMORY SAVER:
     Go to Settings > Performance > Disable "Memory Saver".
     Or add 'youtube.com' to the "Always keep these sites active" list.
  3. AUDIO MANAGEMENT:
     Do not "Mute" the video via the YouTube player (this sometimes blocks 
     the auto-advance feature).
     Instead, lower the volume via the Windows or Linux Volume Mixer 
     so you are not disturbed while playing.
  4. SAFE MODE:
     If the game is highly demanding, detach the YouTube tab into a separate 
     window so it maintains its CPU priority.
